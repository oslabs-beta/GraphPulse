What is GraphPulse?

GraphPulse allows engineers to track GraphQL query performance metrics like latency and depth. Using these metrics, developers can use GraphPulse to run analyses on schema complexity and resolver time execution.

Query Visualization

See the depth and complexity of queryies when executed in real time.

Query Latency Tracking

Track the latency of queries and compare them to past queries.
